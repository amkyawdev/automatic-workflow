Frequently Asked Questions

How do I create my first workflow?

Click "New Workflow" in the Workflows section, give it a name, choose a trigger type, and add your actions. Save and activate to start!

What are the execution limits?

Free tier allows 100 executions per month. Pro plans have unlimited executions.

Is my data secure?

Yes! All data is encrypted at rest. We never share your data with third parties. API keys are encrypted using AES-256.

How do I get an API key for a service?

Each service has different instructions. Generally, you'll need to go to the service's developer portal and create an API key with the appropriate scopes.
